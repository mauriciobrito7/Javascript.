Objeto Location

Propiedades

  • href

    url actual, se puede cambiar para cambiar la página actual

  • host

    dominio:puerto

  • hostname

    dominio

  • port

    puerto

  • protocol

    protocolo(http, https)

  • origin

    protocolo + dominio

  • hash

    #

  • pathname

    ruta interna luego del dominio

Métodos

  • reload()
  • assign(url)
  • replace(url)

    cambia la página sin guardar en history

Objeto History

Este objeto contiene el historial de navegación de la pestaña incluida la página actual

  • Propiedades
    • length

      Cantidad de elementos, incluye la página actual

  • Métodos
    • back()
    • forward()
    • go(position)

      -1 back, 1 forward

Objeto Date

Un objeto que trae las fechas desde el sistema, no de el navegador sino del sistema operativo

  • Instanciar
    • new Date()
    • new Date(year,month,day) meses de 0 a 11
    • new Date(year,month,day,hours,minutes,seconds)
    • new Date(string)
  • Métodos
    • getFullYear()
    • getMinutes()
    • getSeconds()
    • getDate()
    • getDay() días de 0 a 6
    • getMonth() meses de 0 a 11
    • getTime() milisegundos desde el 1 de enero de 1970
    • getTimezoneOffset
